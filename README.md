# Austin Multifamily Intelligence Dashboard

A Streamlit dashboard for analyzing Austin multifamily real estate market data.

## Setup

1. Add `DATABASE_URL` to Streamlit Cloud secrets
2. Deploy from this repo pointing to `dashboard.py`

## Data Sources

- Austin Open Data Portal (Certificate of Occupancy permits)
- CoStar (submarket vacancy, rent, and supply data)
- Neon PostgreSQL (processed data warehouse)
